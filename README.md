# K&R C Programming Exercises

This repository contains selected exercises completed while studying
*The C Programming Language* by Brian W. Kernighan and Dennis M. Ritchie.

## Structure
- Each chapter has its own folder
- Exercises are implemented in C
- Focus areas include:
  - Pointers and memory management
  - File I/O
  - String manipulation
  - Control flow and algorithms
  - Error handling

## Purpose
This repository is for learning and practice purposes only and is not intended
to represent production-level projects.
